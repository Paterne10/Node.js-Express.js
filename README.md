# Node.js-Express.js-
You can find here some fun projects that i built using node environnement with Express.js framework.
# How to run each projects ? 
  1- Run that command bellow to install the node_modules
    1.1- npm -i
  2- Lauch the server by running
    2.1- nodemon file.js
  3- Open the browser and navigate to that address
    3.1- http://localhost:3000/
# Notice
You will notice that i've already push node_modules for some projects. If that doesn't works just remove it and install it on your own.
